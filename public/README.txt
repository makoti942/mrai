The APK is now automatically built by GitHub Actions.

You do NOT need to place an APK file here manually.

The download button on the website fetches the APK directly from
GitHub Releases (built automatically by the CI workflow).

To trigger your first build:
  1. Push the code to GitHub
  2. Go to your repo's Actions tab
  3. The "Build VoiceAgent APK" workflow runs automatically
  4. After it completes, create a GitHub Release
  5. The website will then serve the APK from the release

For local testing, you can still place an APK here and update
index.html to use a local path.
